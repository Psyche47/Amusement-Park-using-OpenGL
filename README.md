### **Design of an Amusement Park using OpenGL**

#### The design of this park includes rides such as,
		1. A Ferris Wheel. 
		2. A Merry-Go-Round.
		3. A Playhouse.
		4. Train Ride.
		5. A Castle 

#### It also includes,
		* A road where cars are passing.
		* Several trees.
		* An ice cream cart
		* A popcorn cart. 
		* The color of the sky also changes according to the time of the day.
